BROODLINE — PLAYABLE BUILD

Windows quick start:
1. Double-click PLAY_BROODLINE.cmd.
2. Your browser will open http://127.0.0.1:8080.
3. Choose BEGIN HUNT, then click the game view to capture the mouse.
4. Keep the server window open while playing. Press Ctrl+C there to stop.

Controls:
WASD move • Mouse aim • Left click fire • R reload • Shift sprint
E interact / buy / rebuild • 1/2/3 or wheel switch • Esc or P pause

The game runs entirely on your computer. No external asset server is required.
See ASSET_LICENSES.md for third-party asset provenance.
