BROODLINE - PLAYABLE LAN BUILD

QUICK START
1. Double-click PLAY_BROODLINE.cmd.
2. Keep the BROODLINE LAN SERVER window open.
3. Your browser opens http://127.0.0.1:8080 automatically.
4. Choose BEGIN HUNT for solo play, or LAN CO-OP to host/join.

HOSTING A SQUAD
1. Choose LAN CO-OP, HOST SQUAD, then CREATE OPERATION.
2. Send the displayed LAN INVITE to players on the same network.
3. Guests open the link, choose JOIN SQUAD, and use the prefilled code.
4. When everyone is ready, choose DEPLOY SQUAD.

The host PC runs the authoritative game. Keep both the browser game and server
window open. If Windows Firewall asks, allow Node.js on private networks. No
port forwarding, cloud account, or dedicated internet server is required.

CONTROLS
WASD move | Mouse look | Left click fire | R/right click reload | Shift sprint
E interact / hold to revive | Q hold to heal / self-revive
1/2/3 or wheel switch weapons | Esc or P field menu | M audio

GAMEPAD
Left/right sticks move/look | RT fire | X reload | A interact/revive
Y heal/self-revive | Bumpers cycle weapons | Menu pause
